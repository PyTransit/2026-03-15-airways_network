Le repo contient deux fichiers :
- ``reseau.txt`` : le résultat est un réseau de type ``hub and spoke`` avec plusieurs hubs interconnectés entre eux
- ``boucle.txt`` : le résultat est une boucle parfaite

NB : Les deux réseaux obtenus sont **connexes**, c'est-à-dire qu'un aéroport est au moins connecté à deux autres.

Pour télécharger les fichiers, appuyer sur le fichier voulu puis sur l'icône de téléchargement en haut à droite.
